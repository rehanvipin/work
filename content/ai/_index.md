# Artificial Intelligence.
I have learnt and built AI applications and frameworks in the past.
However, I haven't documented these experiences before. This is now an attempt at documenting everything.

But why ? Why should I focus on this ? 

I like developing software to which is fun to use and makes people's lives easier.
The capabilities of software, and the way that it can be developed have both become better these past few years.
I would like to use the tools in the most efficient way.

Additionally, I also want to improve these tools. I think they can be more secure, privacy respecting, and performant.

Finding a niche here and improving on it could be a good opportunity.

## Goals
These are some projects and courses that I would like to learn and work on
* Improving accuracy of LLMs.
* Run LLMs in resource constrained environments.
* Enable privacy preserving machine learning.
* Data science. To make data driven decisions.
